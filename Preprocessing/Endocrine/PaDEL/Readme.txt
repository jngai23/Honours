This folder is for processing data from PaDEL Descriptors found at http://www.yapcwsoft.com/dd/padeldescriptor/

The PaDEL-Descriptor application can be run with './run_padel.sh' using data from /Data/Data_Management/folder. This dataset is a cleaned version of the DeepSMILE/SMILE data from the Toxric database that has been processed by 'smilestrip.py'
